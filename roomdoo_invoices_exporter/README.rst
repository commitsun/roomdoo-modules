Exports invoices and their associated payments to a multi-sheet Excel
file.

The report generates three sheets:

- **Invoices**: One row per invoice with fiscal data, amounts, payment
  status and overdue information.
- **Payments**: One row per payment, linked back to its invoice.
- **Summary**: Aggregated totals for invoicing, collections, breakdown
  by payment method, and a control check.

Usage
=====

1. Navigate to **Accounting > Invoices**.
2. Apply the desired filters (date, journal, status, etc.).
3. Select the invoices you want to export — or use *Select All*.
4. Open the **Print** menu and click **Invoices & Payments (Excel)**.
5. The browser downloads the ``.xlsx`` file automatically.

All selected invoices must belong to the same company.
