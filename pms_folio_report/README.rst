Adds a printable report to the **PMS Folio** model, accessible from the
**Print** menu on any folio list or form view:

- **Booking Report (Excel)**: 3-tab ``.xlsx`` file generated via
  ``report_xlsx``:

  - *Summary* – one block per folio with a grey header row, internal
    comments and the list of its reservations.
  - *Reservations* – flat table with one row per folio.
  - *Room Detail* – flat table with one row per reservation; rows are
    colour-coded by folio for visual grouping.

Usage
=====

1. Go to **PMS → Folios** (list or form view).
2. Select one or more folios.
3. Open the **Print** menu (⚙ or top-right printer icon).
4. Choose **Booking Report (Excel)** – downloads a ``.xlsx`` file with 3
   tabs: *Summary*, *Reservations* and *Room Detail*.
