Configuration
=============

* Go to  FastAPI > FastAPI endpoints  and create a new one with the data:
    * Name: PMS API
    * App: PMS API
    * Root path: /pmsApi
    * User: PMS Fastapi
* After the creation push the sync registry button and restart the Odoo server.
